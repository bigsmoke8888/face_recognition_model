Step 1:
1.1Install hbase 1.7.2
In bashrc file add the following
bashrc
     export HBASE_HOME=/home/cloudlab65/hbase-1.7.2-bin/hbase-1.7.2
       export PATH=$PATH:$HBASE_HOME/bin
1.2:
In hbase site.xml add the following 
           <property>
             <name>hbase.rootdir</name>
             <value>file:///home/cloudlab65/hbase</value>
         </property>
         <property>
            <name>hbase.zookeeper.property.dataDir</name>
            <value>/home/cloudlab65/zookeeper</value>
          </property>
1.3:
In hbase-env.sh add the following:
export JAVA_HOME=/usr/lib/jvm/java-8-openjdk-amd64


Step 2:
Create files in home die:
2.1:
mkdir hbase

2.2:
Sudo chmod 777 zookeeper/
sudo chmod 777 hbase

3.1:
start-hbase.sh
